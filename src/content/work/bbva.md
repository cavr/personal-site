## BBVA

Software Engineer at BBVA, one of the largest financial institutions in the world, working on their API marketplace initiative.

### Technical Highlights

- **API Marketplace**: Launched and maintained the API-based marketplace enabling third-party developers to build on BBVA's banking infrastructure
- **Sandbox Development**: Built a sandbox solution for developers to test API integrations safely
- **Multi-country Integration**: Implemented integrations across multiple countries including US, Mexico, and Spain
- **Documentation**: Enhanced API documentation to improve developer experience and testing capabilities

### Impact

Contributed to BBVA's open banking initiative, enabling fintech companies and developers to build innovative financial products on top of banking infrastructure.
